As part of a cleanup of the WorldEditor code we have renamed bigbang to 
worldeditor within the source tree.  We have also renamed some objects to 
reflect this change.  The most noticable name change was renaming the BigBang 
object to WorldManager.  Also, scripts that used the BigBang module now use the
WorldEditor module.

In addition we have moved lots of top-level source files into sub-directories
since it was becoming difficult to find files.  The vcproj file has been
updated to reflect the new sub-directory structure.
