Terrain import/export mode

Key Action
Alt Samples the maximum height of the region selected by dragging the mouse.
Ctrl Samples the minimum height of the region selected by dragging the mouse.
Shift Locks the selected region being dragged to chunk boundaries.
Mouse left
button drag 		Moves or resizes the selected area, depending on whether the mouse is clicked inside the selected area (in which case the move mode is activated) or in the blue border of the selected area (in which case the resize mode is activated).

If Alt is pressed: – See Alt on this page.

If Ctrl is pressed: – See Ctrl on this page.

Mouse right
button drag 		Pans the map.
Mouse scroll
wheel 		Zooms the world map in and out, according to the direction of scroll.
Mouse middle
button click 		Displays the clicked area in the viewport – the top view of the world is abandoned.
