Editing

Key Action
F4 Cycles through object locking modes:
 • Free
 • Terrain
 • Obstacle
This value can also be set by the Object panel's Locking Mode group box.
F6 Cycles through chunk visualisation modes:
 • Normal – Also activated by
 • Chunk borders – Also activated by
 • Pole markers – Also activated by
 • Wireframe – Also activated by
F8 Cycles through coordinate filters:
 • World
 • Local
 • View
This value can also be set by the Object panel's Coordinate System group box.
~ (tilde) Toggles the player walkthru mode.
This mode can also be toggled by or by the Navigation panel's Player Preview Mode group box's Enabled check box.
B Toggles BSP drawing mode.
Same as setting Options tab's, Show BSP field to Custom.
E If the Navigation panel is active and in player walkthru mode:

Increases the camera height for the mode.

M Toggles the drag-on-select mode.
This value can also be set by the Object panel's Drag On Select check box.
Q If the Navigation panel is active and in player walkthru mode:

Decreases the camera height for the mode.

R If a shell is selected:

Recreates the selected shell(s).

V If a shell is selected:

Makes a copy of the selected shell, and links both by a matching portal.
V must be pressed before and during the middle button click.

Ctrl+L Cycles through lighting modes:
 • Standard
 • Dynamic
 • Specular
Shift+F4 Captures a copy of the current scene to bigworld/tools/worldeditor/show<nnn>.bmp, where <nnn> is a sequential number.
