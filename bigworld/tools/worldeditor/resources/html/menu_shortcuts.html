Menu shortcuts

Key Action Menu Item
F1 Opens this page. Help→Shortcuts
Alt+F4 Closes the application. File→Exit
Ctrl+A Selects all objects in the space. Edit→Select All
Ctrl+B Saves selected items as a prefab. Edit→Save Selection As Prefab
Ctrl+D Deselects all selected objects. Edit→Deselect All
Ctrl+H Toggles visibility of all panels. View→Show Panels
Ctrl+K Reloads all chunks. File→Reload All Chunks
Ctrl+R Reloads all texture files. File→Reload All Textures
Ctrl+S Performs a quick save. File→Save
Ctrl+T Saves the selected shell as a template. File→Save Shell As Template
Ctrl+Y Redoes the most recent undone world-changing action. Edit→Redo
Ctrl+Z Undoes the most recent world-changing action.
Only actions that change the world (such as modifying terrain or manipulating objects) are added to the redo list.
Actions like changing brush size or strength, or changing object snap settings are NOT added to the list. 		Edit→Undo
Ctrl+Alt+S Performs a full save. File→Process Data
