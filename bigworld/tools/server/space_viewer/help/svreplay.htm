Basic operation

Left Mouse Button:
Click to select a cell. The selected cell's real and ghost entities are drawn.
Drag to move view surface.

Middle Mouse Button:
"Rubber band" zoom in.

Right Mouse Button:
Same as left click.

Scroll Wheel:
Zoom in / out

PgDown / PgUp
Zoom in / out

Left / Right
Backward/forward 1 second

Up / Down
Backward/forward 60 seconds

Home / End
Warp to beginning/end of log
