The Space viewer module provides a way to telnet to the Python server port of processes that have one. Using the Python console, users can interactively call script commands on a server process, or inspect the state of the server.

Previously, developers would have to use `control_cluster.py pyconsole`, or run telnet from the command line to access this functionality.

1. Usage

Opening the Space viewer module presents the user with a list of running processes that have a Python Console.

Each process has a Connect link, which connects to that process.

The same can be done by accessing ClusterControl module, then selecting Manage Servers link, and in the process Actions drop-down list box selecting Console entry.
