Drag-and-drop

Asset Extension Drop Target Result
Animation .animation Animations panel's, Animation list Adds the dragged animation to the list of animations played for the current model.
Effect .fx Materials Settings panel's Properties table or Effect field Assigns the dragged effect/MFM file to the current material.
Light .mvl Viewport Applies the lighting setup described in the dragged file to the current model.
MFM .mfm Same as Effect.
Model .model Viewport Opens the dragged model/shell for editing.

If Ctrl or Alt is pressed:

Adds the dragged model (with read-only permission) to the viewport, so it can be viewed alongside the current model – this is useful for editing models that are part of a SuperModel

Object Properties panel's Editor Proxy Model field Adds the dragged model as a new LOD level for the current model.
Level of Detail panel's LOD table Sets the dragged model as the editor proxy model.
Shell .model Viewport Opens the dragged shell for editing.
Texture .bmp,
.tga 		Display Settings panel's Choose Floor Texture button Applies the dragged texture to the floor texture.
Materials Settings panel's Properties table's any map field Assigns the dragged texture to the map field.
