Realtime Profiling console (activated by Ctrl+F5)

Key Action
Ctrl+F5 Toggles the console.
Enter Expands/collapses selected element
Esc Exits the console.
Home Scrolls the list to display the first page.
Page Down Moves the selection to the next element in the list.
Page Up Moves the selection to the previous element in the list.
Numpad +
(plus sign) 		Scrolls the screen one line down.
Numpad -
(minus sign) 		Scrolls the screen one line up.
[
(Left square
bracket) 		Moves the selection to the previous element in the list.
]
(Right square
bracket) 		Moves the selection to the next element in the list.
G Draws a graph for the selected element.
