Menu shortcuts

Key Action Menu Item
F1 Opens this page. Help→Shortcuts
Alt+F4 Closes the application. File→Exit
Ctrl+A Adds the selected model to the viewport, so it can be viewed alongside the current model – this is useful when editing a model that is part of a SuperModel.
Please note that added models cannot be changed, only viewed. 		File→Add Model
Ctrl+B Regenerates the model's visibility box using its animations – this is the box used in the client for visibility and shadowing. File→Regenerate Visibility Box
Ctrl+H Toggles the visibility of all panels. View→Show Panels
Ctrl+O Opens the Open dialog box, where you can specify a .model file to load. File→Open Model
Ctrl+R Discards all changes made to the model, and loads its saved version. File→Revert To Saved
Ctrl+S Saves the current model and any other modified files. File→Save
Ctrl+T Reloads from disk all texture files applied to the model – this is useful to refresh modified textures. File→Reload All Textures
Ctrl+Y Redoes the most recently undone model-changing action. Edit→Redo
Ctrl+Z Undoes the most recent model-changing action.
The mouse cursor must be over the viewport for shortcut to be enabled.
Only actions that change the model (such as modifying model's material or adding/removing animations and LODs) are added to the redo list.
Actions like changing any field in Display Settings panel, or changing the camera's angle or height are NOT added to the list. 		Edit→Undo
