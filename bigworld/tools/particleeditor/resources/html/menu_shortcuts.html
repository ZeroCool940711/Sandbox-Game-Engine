Menu shortcuts

Key Action Menu Item
F1 Opens this page. Help→Shortcuts
Alt+F4 Closes the application. File→Exit
Ctrl+H Toggles the visibility of all panels. View→Show Action Selection
Ctrl+O Opens the Open dialog box, where you can specify a new current folder.
This action can also be trigged by the General sub-panel's button. 		File→Open Particle Directory
Ctrl+S Saves the current particle system.
This action can also be triggered by the toolbar button. 		File→Save Particle System
Ctrl+T Reloads textures used by the particle system.
Changes made to textures in third-party applications will not be visible until the particle system is reloaded or this option is selected. 		File→Reload All Textures
Ctrl+Y Redoes the most recent undone action. Edit→Redo
Ctrl+Z Undoes the most recent action. Edit→Undo
