Particle systems

Key Action
F3 Plays the current particle system.
This value can also be set by the toolbar button.
F4 Stop the current particle system.
This value can also be set by the toolbar button.
F5 Cycles between pausing and resuming the current particle system..
This value can also be set by the toolbar button.
Ctrl If clicking on a particle's sub-system check box:

Causes all other sub-subsystems to acquire the opposite value of current sub-system.

Delete Deletes the selected objects.
