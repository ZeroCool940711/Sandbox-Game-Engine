Chapter 36. ual_config.xml

This file configures various aspects of the appearance, layout, and working of the Asset Browser on the BigWorld tools that use it (World Editor, Model Editor, and Particle Editor), and is located under bigworld/tools/<tool_folder>/resources/ual.

Mentioned in document Content Tools Reference Guide's section Asset Browser, Customisation, the tags in this file are documented in bigworld/tools/worldeditor/resources/ual/ual_config.xml.
