Detailed Description

This class is used by BigWorld to implement a Python object that redirects Python's stdout and stderr streams to Dev Studio's output window and a log file which defaults to Python.log. The BigWorld client actually uses an instance of a subclass of this called BWOutputWriter, which also sends stdout and stderr to the BigWorld Python colsole. This is automatically set up by the engine, and neither of these classes nor any instances of them can be accessed via script.
