RemoteEntityMethod Class Reference

RemoteEntityMethod is an actualised class. It is not bound to any module and cannot be directly instantiated.

Parent Classes

PyObjectPlus

Attributes

argumentTypes 
returnValueTypes 

Detailed Description

An instance of RemoteEntityMethod is return from an entity mailbox to allow the calling of a method on a remote machine

Attribute Documentation

argumentTypes

This attribute is a tuple that describes the type of each argument that an entity method expects. Each element in the tuple is a 2-tuple containing the argument's name and a string describing the type of that argument.

returnValueTypes

This attribute is a tuple that describes the type of each return value that an entity method expects. Each element in the tuple is a 2-tuple containing the return value's name and a string describing the type of that return value.
