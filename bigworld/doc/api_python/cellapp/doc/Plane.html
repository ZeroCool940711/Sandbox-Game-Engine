Detailed Description

The Plane class is a 3d plane

A Plane can be created using the Math.Plane() function.

Member Function Documentation

def init( self, p0, p1, p2 ):

This method initialises the plane equation so that the input points lie on it. The points should be in a clockwise order when looked at from the front of the plane. The points must not be colinear.

Parameters:
p0 A Vector3 point on the plane.
p1 A Vector3 point on the plane.
p2 A Vector3 point on the plane.

Returns:
The location of the intersection
