ClientCaller Class Reference

ClientCaller is an actualised class. It is not bound to any module and cannot be directly instantiated.

Parent Classes

PyObjectPlus

Detailed Description

Instances of the ClientCaller class represent entity member functions which a cell application can call on one or more clients. When an object of this type is called it adds a request to call the appropriate function on to the network bundles that are sent to the appropriate clients.

ClientCaller objects cannot be created via python, and are available only through instances which are created by PyClient objects. Further information and an example regarding the use of ClientCaller objects can be found in the PyClient class documentation.
