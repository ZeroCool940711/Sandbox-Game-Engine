Detailed Description

This class is used to keep track of the camera inside the World Editor. World Editor uses different camera types, and you can obtain these cameras by using the WorldEditor.camera function, which will return a type derived from this BaseCamera class.

Attribute Documentation

invert

The invert attribute allows for inverting the direction of movement of the camera when the user is interacting with it.

Type:
Boolean

position

The position attribute determines the translation component of the camera's view matrix.

Type:
Vector3

speed

The speed attribute determines how quick the camera will move when the user is interacting with it (via the WASD keys for example).

Type:
Float

turboSpeed

The turboSpeed attribute determines how quick the camera will move when the user is interacting with it in "turbo" mode (via the WASD keys for example). For information on how to enable turbo mode in the tool, please refer to the tool's help menu items and documentation.

Type:
Float

view

The view attribute determines the view matrix used for the camera.

Type:
Matrix
