Detailed Description

This class inherits from SimpleGUIComponent. It draws a resizable frame in 1 draw call, using a very specific texture layout. Please see the CCM for diagrams and information on how to author the texture map. A new FrameGUIComponent2 is created using GUI.Frame2 function.
