Detailed Description

This is a subclass of Vector4Provider. This provides a Vector4 value that returns the distance in metres between the two given Matrix providers. The distance is returned in all 4 components



Code Example: 
# This creates a Vector4 that returns the distance from the camera
# to the player's head

# import the Math module
import Math

v = Math.Vector4Distance()
v.a = BigWorld.player().model.node("biped Head")
v.b = BigWorld.camera().invViewMatrix()
