Member Function Documentation

def stringDimensions( self, text, multiline, colourFormatting ):

This function returns a 2-tuple containing the dimensions (in pixels) that the specified string will take when rendered to a TextGUIComponent using this font.

Parameters:
text The string to measure the length and height of.
multiline Optional boolean. Whether or not newline characters should be wrapped.
colourFormatting Optional boolean. Whether or not this string is using formatting tags.

Returns:
A 2-tuple of integers, the render width and height of the string in pixels.
