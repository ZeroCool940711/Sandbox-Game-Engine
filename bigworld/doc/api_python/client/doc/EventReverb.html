Detailed Description

A EventReverb provides access to FMOD Event Reverbs.

Attribute Documentation

active

Activates/deactivates this reverb effect within the 3D world.

Type:
Read-write boolean

maxDistance

The minimum distance the listener has to be to this reverb object before the reverb effect is applied at full volume.

Type:
Read-write float

maxDistance

The maximum distance at which the reverb affects the listener. The reverb effect is not audible outside this radius.

Type:
Read-write float

position

The centre position of the reverb. This is overwritten with the output of the source attribute if it has been specified.

Type:
Read-write Vector3

source

The matrix provider which will be used to determine the location of the reverb within the world. This overwrites the position attribute.

Type:
Read-write MatrixProvider
