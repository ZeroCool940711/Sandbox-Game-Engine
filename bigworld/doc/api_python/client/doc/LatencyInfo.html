Detailed Description

The LatencyInfo class is a sub class of Vector4Provider that has an overloaded value attribute. This attribute is overloaded to return a 4 float sequence consisting of ( 1, minimum latency, maximum latency, average latency ) if the client is connected to a server. If the client is not connected to the server then the value attribute returns ( 0, 0, 0, 0 ).

Call the BigWorld.LatencyInfo function to create an instance of a LatencyInfo object.
