Detailed Description

The PyMetaParticleSystem can contain several PyParticleSystem objects, allowing them to function as a single object. As a PyAttachment, the PyMetaParticleSystem can be attached to other BigWorld objects.

A new PyMetaParticleSystem is created using Pixie.MetaParticleSystem function.

Member Function Documentation

def addSystem( self, particle_system ):

This function adds a ParticleSystem to the MetaParticleSystem container.

Parameters:
particle_system The ParticleSystem to add to this container.

def clear( self ):

This function calls the clear method of each contained ParticleSystem (effectively removes all particles from all contained ParticleSystems).

def duration( self ):

This function looks for a sink in each contained ParticleSystem and returns the maximum age of each particle system, returns -1 if no sinks are found

def force( self, num ):

This method forces each ParticleSystem contained within the MetaParticleSystem to forcibly create num units of particles. The number of particles created by each unit are dependant upon the individual ParticleSystems forcedUnitSize in an attached source ParticleSystemAction (see SourcePSA).

Parameters:
num An integer representing the number of units to force on the SourcePSA of all contained ParticleSystems.

def load( self, name ):

This function loads a saved MetaParticleSystem definition from an XML file.

Parameters:
name Name of the XML file to load.

def nSystems( self ):

This function returns the number of ParticleSystems contained within this MetaParticleSystem.

def removeAllSystems( self ):

This function removes all ParticleSystems from the MetaParticleSystem container.

def removeSystem( self, particle_system ):

This function removes a ParticleSystem from the MetaParticleSystem container.

Parameters:
particle_system Either a string containing the name of the ParticleSystem or a reference to the ParticleSystem to be removed.

def render( self ):

Draws all ParticleSystems contained within this MetaParticleSystem. You do not need to call this method.

def save( self, name ):

This functions saves a MetaParticleSystem definition to an XML file.

Parameters:
name Name of the XML file to save to.

def system( self, particle_system ):

This function returns a reference to a ParticleSystem within the container, or None if no matching ParticleSystem is found.

Parameters:
particle_system String containing the name of the ParticleSystem to return.

def update( self, dt ):

Updates all particle systems contained within this MetaParticleSystem by the specified time slice dt. You do not need to call this method.

Parameters:
dt Time interval to update particle system by (in seconds).
