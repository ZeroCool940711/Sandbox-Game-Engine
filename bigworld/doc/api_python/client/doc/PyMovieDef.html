Detailed Description

This class wraps Scaleform's Movie Definition and provides lifetime control and python access to its API.

A Movie Definition contains the resources for an .swf file, and allows creation of instanced views of the movie.

Movie Definitions can also be loaded for the sole purpose of providing fonts for other movies and for flash text components.
