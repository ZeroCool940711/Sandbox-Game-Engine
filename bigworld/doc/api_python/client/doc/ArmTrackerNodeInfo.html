Detailed Description

This is a subclass of BaseNodeInfo. This describes a two bone arm system used to either reach a specific direction or point in a specific direction. The values are all provided to the factory method used to create this (BigWorld.ArmTrackerNodeInfo).

Refer to the Tracker class documentation for an overview of the tracker system and related code samples.
