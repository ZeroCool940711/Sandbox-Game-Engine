Detailed Description

This class is a utility class designed to help implement collisions on dynamic skeletal objects.

It has a list of BoxAttachment objects, and provides a C++ and python interface to coordinate collision testing for that list as a whole.

def doCollide( self, start, direction ):

Attempts to collide with this collider. Tests the collection of colliders against the ray that begins at the position defined by the "start" argument, and goes all the way down to the far plane in the direction defined by the "dir" argument.

Parameters:
start the start point of the collision ray
direction the direction of the collision ray.

Returns:
True if a hit was detected, false otherwise. Check the impactCollider, impactPoint and impactReflection attibutes for details on the intersection detected.
