Detailed Description

An Orbitor is a Motor that moves a model towards a target (MatrixProvider) and when within a certain distance starts to move towards an orbiting path around the target. Additionally an optional proximity callback can be set for the Orbitor to call once the model it is moving gets within a specified (proximity) distance of its target.

A new Orbitor Motor is created using BigWorld.Orbitor function.
