Detailed Description

This class is used to control the weather in the clients world.

The weather object is used to set the wind speed and wind gustiness. The wind affects the rate at which clouds move across the sky, and the pertubations of detail objects such as grass.

Member Function Documentation

def windAverage( self, xWind, zWind ):

This function sets the average velocity of the wind, specified as x and z coordinates. The wind on average blows at this velocity but deviates around it by a random amount which is capped at windGustiness.

The wind determines the direction and speed that clouds move across the sky. This is important, because when weather systems are changed the changes appear 1000m upwind of the user, and are blown across the sky at a rate specified by the wind velocity.

If the wind is changed, then existing clouds continue with their existing velocities, and new clouds are produced with the new velocity, so that the weather gradually switches to blow from a different direction.

The wind velocity is also used to perturb the detail objects, allowing for swaying grass and the like.

Parameters:
xWind a float. The x component of the winds average velocity.
zWind a float. The z component of the winds average velocity.

def windGustiness( self, gust ):

This function specifies the gustiness of the wind. This is the largest magnitude by which the actual wind speed can vary from that specified by windAverage, as it varies randomly.
