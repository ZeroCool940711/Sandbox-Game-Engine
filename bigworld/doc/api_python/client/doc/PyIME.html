Detailed Description

Provides access to the current IME state for the client process. This cannot be directly instanciated and is accessed via the BigWorld.ime singleton object.

Attribute Documentation

candidates

Contains a list of possible candidates based on the keystrokes entered by the user.

Type:
Read-Only list of Unicode strings

candidatesVertical

Determines whether ot not the candidate list is horizontal or vertical for the current language.

Type:
Read-Only boolean

candidatesVisible

Determines whether or not the candidate list should be displayed to the user.

Type:
Read-Only boolean

composition

Contains the unicode string representing the current composition entered by the user.

Type:
Read-Only Unicode string

compositionAttr

Of the same length as the composition string, this specifies the current conversion attributes for each character in the string.

Possible values are: 

ATTR_INPUT					= 0
ATTR_TARGET_CONVERTED		= 1
ATTR_CONVERTED				= 2
ATTR_TARGET_NOTCONVERTED	= 3
ATTR_INPUT_ERROR			= 4
ATTR_FIXEDCONVERTED			= 5

Type:
Read-Only Array of integers

compositionCursorPosition

Index into the composition string representing where the input cursor is currently located.

Type:
Read-Only int

currentlyVisible

If this is true, then there are IME components with which the user are currently interacting (e.g. composition string and/or candidate list).

Type:
Read-Only boolean

enabled

Controls whether or not IME is enabled for the client. For example, you would set this to True when an IME enabled edit control comes into focus and then set it to False when it loses focus.

Type:
Read-Write boolean

language

Reflects the current language selected by the user. Supported languages are are: 

"NON_IME"	- A language that does not require an IME.
"CHS"		- Chinese (Simplified)
"CHT"		- Chinese (Traditional)
"JAPANESE"	- Japanese
"KOREAN"	- Korean

Type:
Read-Only string

reading

Contains the unicode string representing the current reading string entered by the user.

Type:
Read-Only Unicode string

readingVertical

Determines whether ot not the reading string is horizontal or vertical for the current language.

Type:
Read-Only boolean

readingVisible

Determines whether or not the reading string should be displayed to the user.

Type:
Read-Only boolean

selectedCandidate

Index into the candidate selection list of the currently highlighted candidate.

Type:
Read-Only int

state

Reflects the current state of the IME. Possible values are: 

"OFF"		- The IME is turned off.
"ON"		- The IME is active.
"ENGLISH"	- The IME is active, but is in alpha numeric mode.
