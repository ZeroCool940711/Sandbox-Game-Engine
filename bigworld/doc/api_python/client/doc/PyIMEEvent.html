Detailed Description

This class is used to notify the scripts when an IME state change has occured. It has a set of flags to determine which properties on the global PyIME object have changed.

Attribute Documentation

candidateSelectionChanged

If this is True, then the PyIME.selectedCandidate attribute has changed.

candidatesChanged

If this is True, then the PyIME.candidates attribute has changed.
