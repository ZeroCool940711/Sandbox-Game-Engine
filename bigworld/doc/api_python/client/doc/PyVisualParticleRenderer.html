Detailed Description

VisualParticleRenderer is a ParticleSystemRenderer which renders each particle as a visual object. Any visual can be used, although the TintShader will only work on visuals that use effect files that use MeshParticleTint.

Attribute Documentation

visual

This is the name of the visual particle type to use for the particles. Default value is "".
