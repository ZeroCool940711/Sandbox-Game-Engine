Detailed Description

This is a subclass of Vector4Provider. This provides a Vector4 value that returns the translation part of the source Matrix.



Code Example: 
# This creates a Vector4 that returns the position of the players
# head node.

# import the Math module
import Math

v = Math.Vector4Translation()
v.source = BigWorld.player().model.node("biped Head")
