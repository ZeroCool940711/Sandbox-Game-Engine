Detailed Description

An EventCategory is a wrapper for an FMOD::EventCategory, and is a virtual folder used to organize event data into generic categories such as music. Allows volume and pitch changes to be applied to all events within the group.

For more information about event categories and how they are used, please see the FMOD Designer API documentation and the FMOD Designer User Manual, both available from www.fmod.org.
