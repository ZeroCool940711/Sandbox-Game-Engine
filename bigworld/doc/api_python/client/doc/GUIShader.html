Detailed Description

This is the abstract base class for all GUI shaders. A GUI shader is something that you apply to a GUI component, and it modifies the way in which the component is drawn. It also modifies the way that components children are drawn.

GUIShaders are added to components using the SimpleGUIComponent.addShader method and removed using the SimpleGUIComponent.delShader method. All other gui components inherit these from SimpleGUIComponent.
