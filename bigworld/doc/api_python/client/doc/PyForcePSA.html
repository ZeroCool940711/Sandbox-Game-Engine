Detailed Description

PyForcePSA is a PyParticleSystemAction that applies a constant acceleration to particles in a particular direction.

A new PyForcePSA is created using Pixie.ForcePSA function.

Attribute Documentation

vector

This is a vector describing the direction and magnitude of the acceleration to be applied to the particles. The default value is (0.0, 0.0, 0.0).

Type:
Sequence of 3 floats.
