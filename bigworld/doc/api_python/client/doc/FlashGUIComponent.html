Detailed Description

The FlashGUIComponent is a GUI component used to display a flash movie on the screen. It inherits from SimpleGUIComponent.

It delegates all drawing, input handling to flash. It can have an associated python script, like any gui components.

The flash movie will draw within the bounds of this gui component.

A new FlashGUIComponent is created using the GUI.Flash function.

For example: 
mvDef = Scaleform.createMovie( "scaleform/flash_ui.swf" )
g = GUI.Flash( mvDef.createInstance() )
GUI.addRoot( g )
setattr( g.movie, "_root.actionScriptVariable", "hello world" )
g.movie.invoke( "_root.OpenMenuLevel2" )


This example creates and displays a FlashGUIComponent, sets and action script text variable to "hello world" then calls an action script function via the invoke() call.
