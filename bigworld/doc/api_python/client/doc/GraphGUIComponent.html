Detailed Description

The GraphGUIComponent is a GUI component used to display a line graph of a Vector4Provider on the screen.

A new GraphGUIComponent is created using GUI.Graph function.

Attribute Documentation

frequency

This specifies the rate at which the graph will be updated. If frequency is less than 0, the update rate will be in frames (ticks). If it is greater than 0, it will be in seconds. Defaults to 0.033

Type:
float

gridColour

Specifies the colour to use when drawing the background grid. Defaults to (255, 255, 255, 255).

input

This stores the information to be displayed on the screen. The Vector4Provider is represented as 4 line graphs, one for each of the x,y,z,w components.

Example: 
import GUI
import Math
g = GUI.Graph( "" )
GUI.addRoot(g)
g.input = Math.Vector4Product()
g.input.a = Math.Vector4LFO()
g.input.b = ( 1.0, 0.8, 0.4, 0.2 )
g.frequency = 0.015
g.nPoints = 100
g.size = (1.5, 1.5)

Type:
Vector4Provider

lineColours

This list configures the colours used to display each line in the graph (maximum of four lines).

Type:
list of Vector4

maxY

Specifies the maximum height that the graph will be displayed on the screen. Value should be between minY and 1.0 (top of screen). Defaults to 1.0

Type:
float

minY

Specifies the minimum height that the graph will be displayed on the screen. Value should be between maxY and 0.0 (bottom of screen). Defaults to 0.0

Type:
float

nPoints

This stores the number of points that will be displayed in the graph. Defaults to 25.

showGrid

Toggles the display of a background grid. Defaults to False.
