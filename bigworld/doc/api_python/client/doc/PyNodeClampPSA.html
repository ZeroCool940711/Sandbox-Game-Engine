Detailed Description

PyNodeClampPSA is a PyParticleSystemAction that locks a particle to the Node the particle system is attached to. Any motion or change in position of the Node is also appied to the particles in the particle system.

A new PyNodeClampPSA is created using Pixie.NodeClampPSA function.

Attribute Documentation

fullyClamp

If set to true, the particles are clamped directly to the particle system position. If false, the particles are moved relatively each frame by the amount the particle system moved by.

Type:
Bool
