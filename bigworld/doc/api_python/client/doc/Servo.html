Detailed Description

The Servo class is a Motor that sets the transform of a model to the given MatrixProvider. As the MatrixProvider is updated, the model will move.

Attribute Documentation

signal

This attribute is the MatrixProvider that sets the model's transform. As this MatrixProvider is updated, the model will be transformed accordingly.
