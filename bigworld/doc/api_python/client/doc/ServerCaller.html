Detailed Description

These objects are automatically created by BigWorld to populate a PyServer object. Each ServerCaller object provides the link to a single method on the component that the PyServer object represents (ie, Cell or Base). These methods are defined in the .def file and can only exist if the server component exists. A ServerCaller object is accessed by invoking the method that it represents in the PyServer object, eg;

self.cell.method( args )

where self is an Entity, cell is the PyObject created by BigWorld (which is not available if there is no Cell component for this Entity) and method( args ) is the ServerCaller object that represents the method on the Cell that was defined in this Entity's .def file
