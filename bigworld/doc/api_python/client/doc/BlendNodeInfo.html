Detailed Description

This is a subclass of BaseNodeInfo. Instances of this class are used by the Tracker system to blend between BaseNodeInfo objects over 0.3 seconds. This is automatically done whenever the nodeInfo or directionProvider properties of a Tracker are changed. Instances of this class cannot be created or accessed via script.
