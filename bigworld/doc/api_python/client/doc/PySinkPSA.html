Detailed Description

PySinkPSA is a PyParticleSystemAction that destroys particles within a ParticleSystem. Particles can be destroyed if they are over a certain age or under or over a particular speed.

A new PySinkPSA is created using Pixie.SinkPSA function.

Attribute Documentation

maximumAge

Particles whose time in existence is greater then maximumAge are removed from the ParticleSystem. Default -1.0. This attribute is ignored if value is < 0.

Type:
Float.

minimumSpeed

Particles with a speed of less than minimumSpeed are removed from the ParticleSystem. Default -1.0. This attribute is ignored if value is < 0.

Type:
Float.
