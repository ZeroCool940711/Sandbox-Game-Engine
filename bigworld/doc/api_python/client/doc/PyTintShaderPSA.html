Detailed Description

PyTintShaderPSA is a PyParticleSystemAction that tints a particle according to its age. The initial colour of a particle is ( 0.5, 0.5, 0.5, 1.0 ) in RGBA at time 0 and tinting occurs linearly from one tint to the next tint over the time interval between which they are set. Tints can be set at particular times using the addTintAt function.

A new PyTintShaderPSA is created using Pixie.TintShaderPSA function.

Member Function Documentation

def addTintAt( self, time, colour ):

Adds a colour (RGBA) to blend the tint to at a particular time. Particle colour gets changed linearly between each tint and the previous tint. You can think of the tint like a key frame for an animation.

Parameters:
