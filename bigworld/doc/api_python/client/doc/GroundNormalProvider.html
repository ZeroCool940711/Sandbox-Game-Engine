GroundNormalProvider is an actualised class. It is not bound to any module and cannot be directly instantiated.

Detailed Description

This class implements a Vector4Provider to the ground normal of an AvatarDropFilter. It does this by maintaining a weak reference to the filter instance.

Attribute Documentation

avatarDropFilter

This attribute returns the AvatarDropFilter weakly reference by the provider or None if the filter has been destroyed.
