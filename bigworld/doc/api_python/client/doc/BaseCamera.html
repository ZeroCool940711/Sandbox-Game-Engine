Detailed Description

This class is the abstract base class for all cameras. It supplies basic functionality for setting a camera to a particular matrix, for updating and rendering the camera, and for reading the camera's position and direction.

There are three subclasses, CursorCamera, FlexiCam and FreeCamera.
