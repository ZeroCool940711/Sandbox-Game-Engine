Detailed Description

PointSpriteParticleRenderer is a ParticleSystemRenderer which renders each particle as a point sprite.

Texture animation (rolling uv) is not supported, nor are particle sizes on the screen > 64 pixels.

This particle renderer is the quickest to draw, and uses the least amount of graphics memory ( push buffer ).

A new PyPointSpriteParticleRenderer is created using Pixie.PointSpriteParticleRenderer function.
