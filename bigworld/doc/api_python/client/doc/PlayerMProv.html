Detailed Description

This class inherits from MatrixProvider, and can be used to access the position of the current player entity. Multiple PlayerMProvs can be created, but they will all have the same value.

There are no attributes or functions available on a PlayerMProv, but it can be used to construct a PyMatrix from which the position can be read.

A new PlayerMProv is created using BigWorld.PlayerMatrix function.
