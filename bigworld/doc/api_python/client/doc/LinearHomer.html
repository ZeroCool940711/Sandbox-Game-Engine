Detailed Description

A LinearHomer is a Motor that accelerates a model towards a target (MatrixProvider). Additionally an optional proximity callback can be set for the LinearHomer to call once the model it is moving gets within a specified (proximity) distance of its target. It has a different set of features to a Homer Motor.

A new LinearHomer Motor is created using BigWorld.LinearHomer function.

The attribute proximityCallback is used to assign a callback function to the LinearHomer Motor that is triggered once the model that the LinearHomer is moving gets within "proximity" distance of "target". Default is None.
