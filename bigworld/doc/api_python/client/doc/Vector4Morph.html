Detailed Description

This is a subclass of Vector4Provider. This provides a Vector4 value that morphs from the current value to the target value, over the specified duration. You can query its current value using the value attribute.



Code Example: 
# This creates a Vector4 that smoothly changes from value to value.

# import the Math module
import Math

v = Math.Vector4Morph()
v.duration = 2.0

print Math.Vector4(v)
# '0, 0, 0, 0'

v.target = (4,4,4,4)
BigWorld.callback( 1.0, partial( checkValue, v ) )

def checkValue( v ):
	print Math.Vector4(v)
	# '2, 2, 2, 2'

	v.target = (2,3,1,9)
	v.time = v.duration
	print v.current
	# '2, 3, 1, 9'
