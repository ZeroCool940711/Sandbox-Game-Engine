Detailed Description

A PyFilterQuad can be used by a PyPhase object, and provides the geometry for the phase. The FilterQuad performs n-taps on the source material's textures, in groups of 4. If you specify more than 4 taps, then the filter quad will draw using n/4 passes. For these operations, you should make sure the material you use is additive, so the results can accumulate.
