Detailed Description

This is the Bots component of the Distributed Entity Model. It is responsible for handling an automated mimicing of similar Client entities. This includes interacting with other entities, and responding to updates from the server.

For more information refer to the Server Operations Guide, chapter Stress Testing With Bots.
