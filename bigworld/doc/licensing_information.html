BigWorld External Libraries Licensing issues

The BigWorld Technology incorporates multiple external libraries with multiple licenses. In order to comply with the different licensing requirements of these libraries the following files are provided:

  • The development_and_deployment_licensing.txt file contains Licenses of external libraries included in the BigWorld server, server tools and client tools components.

  • The game_client_licensing.txt file contains licenses of external libraries included in the BigWorld game client. This license file should also be included in the documentation and/or other materials provided with any game which is created by BigWorld's customers based on the BigWorld client.
